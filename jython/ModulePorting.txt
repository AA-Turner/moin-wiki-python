= Porting Standard Library Modules =

This page is for tracking which standard library modules have been updated from CPythonLib (2.7), patched and committed.

Some modules have patches generated by genpatches.py. Some of those patches failed to apply and will have to be patched manually.
Others don't even have proper patches because the genpatches.py script was unable to locate the last CPython synchronization changeset automatically.
