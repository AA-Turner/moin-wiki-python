== Python Community Server ==

The Python Community Server is a reimplementation of the [[http://rcs.userland.com/|Radio Community Server]] in Python. It can be used to host weblogs that are built with [[http://radio.userland.com|Radio Userland]], [[PyDS]] or [[http://www.myelin.co.nz/bzero/|bzero]] or any other client that implements the XML storage system.

It is a Sourceforge project at http://pycs.sourceforge.net/
