This page is a collection point for people to post links to IDLE extension modules that they have written themselves.

[[http://idlex.sourceforge.net/extensions.html|IdleX]] provides a large collection of extensions to improve IDLE. 

Alias: enhances IDLE's command-line with an alias facility and more. Download @ [[lwickjr]].

Edit: an edit command for Alias that attempts to invoke the IDLE editor on the source code of the object referenced on the command line. Download @ [[lwickjr]].
