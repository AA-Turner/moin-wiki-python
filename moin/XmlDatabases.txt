== BaseX ==

BaseX is a very fast and powerful, yet light-weight XML database system and XPath/XQuery processor, including highly conformant support for the latest W3C Update and Full Text Recommendations. Bindings for python are available.

http://www.basex.org

http://docs.basex.org/wiki/Clients
