This is a list of web log("blog") software written in Python. Please feel free to add to the list or add details.

To start a feature comparison, use the following keys:
c = comments[[BR]]
h = caching[[BR]]
k = pingback / traceback[[BR]]
m = markup support[[BR]]
n = notifications[[BR]]
p = plugin architecture[[BR]]
r = RSS feed[[BR]]
t = topics / categories[[BR]]

== Tools ==

 * [http://pyljclient.sourceforge.net/ pyLJclient]
  * wxPython LiveJournal client

 * [http://pyblogger.wordpress.com pyblogger]
  * wrapper for the Blogger API

 * [http://www.drbeat.li/py/plagg plagg]
  * a RSS/Atom aggregator for (Py)Blosxom
