**Project Info**

Bazaar is a distributed version control system that Just Works and
adapts to the workflows you want to use. It is easy to use, fast,
efficient and lightweight. Some interesting  features include perfect
rename support, bound branches, an open architecture that supports
multiple storage formats and plugins. Bazaar is ideal for open source
projects and distributed development within commercial organizations.

Project URL: http://bazaar-vcs.org/

Sprint Leader: John Arbash Meinel

Downloads and source code repository: http://bazaar-vcs.org/Download

Documentation: http://bazaar-vcs.org/Documentation

Mailing list instructions: http://bazaar-vcs.org/BzrSupport

Developer guide: http://doc.bazaar-vcs.org/bzr.dev/en/developer-guide/HACKING.html.
