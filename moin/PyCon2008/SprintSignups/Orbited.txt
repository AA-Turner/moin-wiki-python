**Project Info**

Orbited is an HTTP daemon that is optimized for long-lasting comet connections.
It is designed to be easily integrated with new and existing applications. 
Orbited allows you to write real-time web applications, such as a chat room or 
instant messaging client, without using any external plugins like Flash or Java.

Project URL: http://www.orbited.org
