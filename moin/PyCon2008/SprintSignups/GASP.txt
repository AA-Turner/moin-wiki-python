**Project Info**

Graphics API for Students of Python (GASP).  GASP provides a simple, 
procedural graphics API for beginning students using Python.  GASP is 
built on top of pygame, and is designed to lead students into full use 
of the python and object oriented programming.

Project URL: https://edge.launchpad.net/gasp

Sprint Leader: Jeff Elkner 

Instructions and new contributor information: https://edge.launchpad.net/gasp-code
