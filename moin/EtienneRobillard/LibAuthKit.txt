Unofficial branch/port of !AuthKit. 

Source code is included in the [[http://www.isotopesoftware.ca/software/django-hotsauce/|DjangoHotSauce]] distribution. (extras/libauthkit)



=== Features ===

 * OAuth2 framework support (requires oauthlib)
 * Schevo database support (requires libschevo)
 * HTTP cookie authentication/authorization (unstable)
