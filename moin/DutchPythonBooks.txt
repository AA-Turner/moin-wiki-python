This page lists Python books written in Dutch.

Deze pagina geeft een lijst van Nederlandstalige boeken over Python. Aangezien de meeste Nederlanders het Engels goed beheersen, zijn er niet veel Nederlandstalige boeken beschikbaar.

----

'''De Programmeursleerling: Leren coderen met Python 3''' 
 Pieter Spronck

[[http://www.spronck.net/pythonbook/dutchindex.xhtml|Eigen publicatie]]
Gratis beschikbaar als PDF,
~400 pagina's (2016)

Dit boek is een Python 3 cursus voor beginners, gericht op middelbare scholieren en studenten voor wie programmeren nieuw is. Het behandelt alle imperatieve en object geori&euml;nteerde aspecten van Python, en gaat ook in op een aantal onderwerpen daarbuiten. Het boek is ruim voorzien van opgaves en oefeningen, inclusief antwoorden. Code is beschikbaar.

----

'''Leren Programmeren in Python'''
 Tom Verhoeff

[[http://www.win.tue.nl/~wstomv/edu/python/python1.html|Web publicatie]]

Dit is een Python 2 cursus, ge-update tot en met Python 2.2.3, beschikbaar gesteld via de TU Eindhoven.
