== Bücher ==

 * [[https://dpunkt.de/produkt/routineaufgaben-mit-python-automatisieren-2/|Routineaufgaben mit Python automatisieren]] Die deutsche Übersetzung () des Buches "Automate the Boring Stuff with Python" von Al Schweigart in der 2. Auflage (jetzt für Python 3) ist eine Python-Einführung am Beispiel von alltäglichen Aufgaben eines Computernutzers. Al Schweigart stellt die [[https://automatetheboringstuff.com/|englische Originalausgabe]] kostenlos zum online-Lesen zur Verfügung. [[https://dpunkt.de/|dpunkt.verlag]].

 * [[http://openbook.rheinwerk-verlag.de/python/|Python 3 Das umfassende Handbuch]] von Johannes Ernesti und Peter Kaiser, als Openbook aus dem [[https://www.rheinwerk-verlag.de/|Rheinwerk Verlag]]. Alternativ auch als Buch/eBook gegen Bezahlung ;-)

 * [[https://oreilly.de/produkt/python-von-kopf-bis-fuss/|Python von Kopf bis Fuß]] von Paul Barry ist ein Buch für Einsteiger. [[http://www.oreilly.de/|O'Reilly Germany]].

 * [[https://oreilly.de/produkt/programmieren-lernen-mit-python/|Programmieren lernen mit Python]] von Allen B. Downey ist an Einsteiger gerichtet. Die 2. Auflage behandelt Python 3.4.

 * [[http://cito.github.io/byte_of_python/|A Byte of Python (deutsche Übersetzung)]] ist die deutsche Übersetzung von Swaroop C Hs [[http://www.swaroopch.com/notes/Python|A Byte of Python]] für Python 2.x.

 * [[https://www.rheinwerk-verlag.de/einstieg-in-python-ideal-fuer-programmiereinsteiger/|Einstieg in Python]], von Thomas Theis, ist ein deutsches Buch für Anfänger. [[https://www.rheinwerk-verlag.de/|Rheinwerk Verlag]].

 * [[https://oreilly.de/produkt/python-kurz-gut/|Python kurz & gut]] ist eine Übersetzung von "Python Pocket Reference" (Mark Lutz) von Stefan Fröhlich und ist auf dem Stand von Python 2.7 und 3.4. [[http://www.oreilly.de/|O'Reilly Germany]].

 * [[http://www.alan-g.me.uk/tutor/german/index.htm|Programmieren lernen]] ist eine Übersetzung (durch Bruno Schäfer) von Alan Gaulds [[http://www.alan-g.me.uk/l2p/|Learning to Program]].

 * [[http://python4kids.net|Python für Kids]] - Gregor Lingls Buch, das Kinder in der Mittel- und Oberstufe an Python heranführt. Die letzte, 4. Auflage benutzt Python 3.1.

 * [[http://www.oreilly.de/catalog/pythonxmlger/|Python & XML]] ist eine Übersetzung von "Python and XML" (Christopher A. Jones & Fred L. Drake, Jr.) von Dinu Gherman.  [[http://www.oreilly.de/|O'Reilly Germany]].

 * [[https://www.hanser-fachbuch.de/buch/Einfuehrung+in+Python+3/9783446452084|Einführung in Python 3]] ist ein Buch von [[http://www.bklein.de|Bernd Klein]], was besonders geeignet ist für Programmieranfänger, aber auch für Umsteiger von anderen Programmiersprachen. [[http://www.hanser-fachbuch.de|Hanser Verlag]].

 * [[https://oreilly.de/produkt/hitchhikers-guide-fuer-python/|Hitchhiker’s Guide für Python]] Kenneth Reitz und Tanya Schlusser befasst sich mit Python Best Practices. [[http://www.oreilly.de/|O'Reilly Germany]].
