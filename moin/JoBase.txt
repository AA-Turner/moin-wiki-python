= JoBase =

JoBase is a really fast Python game library for beginner coders.
It is written pure C and uses the GLFW library.
It also has an integrated Chipmunk physics engine.
Please visit the official website at [[https://jobase.org|jobase.org].

JoBase has been rewritten in JavaScript so that it can run in any major browser!
Feel free to use the online editor at [[https://jobase.org/Browser|jobase.org/Browser]].
