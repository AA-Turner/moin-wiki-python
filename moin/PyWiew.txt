["PyWiew"] is an image viewer/editor written entirely in [http://python.org Python] using [http://www.wxPython.org wxPython] for GUI and [http://www.pythonware.com/products/pil/index.htm Python Imaging Library] for image manipulation. 

This project is meant mainly as a tutorial for [http://python.org Python] beginners as well as to showcase the many capabilities of [http://www.wxPython.org wxPython]. It has basic image viewing capabilities (supported by wxPython/PIL) and image saving/filter/effects capabilities.
