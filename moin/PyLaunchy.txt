= PyLaunchy =
[[http://pylaunchy.sourceforge.net|PyLaunchy]] is an extension to [[http://www.launchy.org|Launchy's]] plugin system, that allows a user to develop Launchy plugins in the [[Python]] programming language.

[[http://www.launchy.org|Launchy]] is a free open source application launcher for GNU/Linux and Windows, written in C++ and uses QT. The application can be extended by writing C++ plugins

PyLaunchy provides a framework for writing Launchy plugins as Python scripts. It uses [[boost.python]] for C++ bindings and PyQt4 for integration with QT.
