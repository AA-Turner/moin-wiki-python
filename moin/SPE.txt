SPE is an open-source, multi-platform python IDE with [http://www.wxpython.org wxPython] interface. It has auto indentation & completion, call tips, syntax coloring & highlighting, uml viewer, class explorer, source index, auto todo list, sticky notes, pycrust shell, file browsers, drag&drop, context help, [http://www.blender.org Blender] support, ... Spe ships with [http://www.digitalpeers.com/pythondebugger rpdb2] (remote encrypted debugger), [http://wxglade.sourceforge.net/ wxGlade] (gui designer), [http://pychecker.sourceforge.net/ PyChecker] (source code doctor) and [http://project5.freezope.org/kiki Kiki] (regex console).

See http://pythonide.stani.be
