''''We are looking forward to the host who makes PyCon India 2010 bigger and better than ever!''''

''Note: The Indian Python Software Society is only interested in conference proposals that are led by either Python communities  or educational institutions who are active in the Python community, such as user groups, Python consulting firms, institutional Python users, etc.''

== Schedule ==
PyCon India 2010 is scheduled to be held on Sptember 25 – 26, 2010.

== Introduction ==
This document is the call for proposals to host Python Conference 2010.  This document contains some more specific guidance on the elements of a conference proposal, conference requirements, locations, and the schedule for the Python Conference 2010 selection process.

First Python conference has been held in IISc in Bangalore.

== Conference Vision and Goals ==
Python conferences are the annual gathering of the India Python software community. Python conferences are an opportunity for the vast and diverse Python community to forge new relationships and catch up with old friends, learn from each other, and recharge the wellsprings of creativity and innovation that power the Python community process.

A great Python conference will offer for a of attendees.

== Required elements of a Python Conference hosting proposal ==
 * Information about your proposed venue: capacity, rooms, multimedia facilities/support, etc.
 * Plan for providing reliable wireless internet access for conference attendees
 * Information about logistics, accommodations, etc in your proposed location ( and adjoining areas).
 * Team of volunteers and a event co-ordinator (preferably a senior staff member) who will work with the Python community in preparation and successful running of the conference.
 * Conference will be branded in all PR activities including official web site as  “Indian Python Software Society in association with [Host name] ”

== Python conference requirements ==
 * Venue must be able to accommodate 600 people
 * Venue must have plenary room capable of accommodating all attendees, plus breakout rooms for 2-3 tracks and informal gathering space
 * Reliable wireless access capable of accommodating all conference attendees simultaneously
 * Appropriate audio-visual support (e.g. sound, projectors, etc.)
 * Optional but preferred, catering facility ( participants will pay for their food)

== Suggested conference design ==

We strongly encourages prospective conference host to include the following elements in their conference design
 * 2 days of conference time
 * 1 day before the conference for  preparation of the conference.
