== Historical Records ==
These tutorials and resources are for Qt 3:

 * [[http://www-128.ibm.com/developerworks/linux/library/l-qt/?ca=dgr-linuxw02QTPyQT|IBM Qt/PyQt tutorial]] - a toy application is developed, first with Tkinter, then with !PyQt.
 * [[http://www.cs.usfca.edu/~afedosov/qttut/|PyQT-tutorial]] - uses Qt Designer, very good for beginners.
 * [[http://wiki.python.org/moin/JonathanGardnerPyQtTutorial|Jonathan Gardner's PyQt tutorial]] - writing a GUI for the `at` utility.
 * [[http://www.commandprompt.com/community/pyqt|GUI Programming with Python: QT Edition]] - A very complete document about !PyQt development (also mentioned in [[PyQt/Books|Books]])
