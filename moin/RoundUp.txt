RoundUp is a simple-to-use and -install issue-tracking system with
command-line, web and e-mail interfaces. It is based on the winning design
from KaPingYee in the SoftwareCarpentry "Track" design competition.

It's a SourceForge project, with the lead developer being RichardJones. You can visit the project page at http://roundup.sf.net/ for more information


== RoundUp for python.org ==
'''RoundUp''' is used for tracking Python bugs at python.org  The following pages accumulate various information about it's installation and usage specifics in Python development.
