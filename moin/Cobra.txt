= Cobra =

The Cobra Programming Language: http://cobra-language.com/

eWEEK publicity: [[http://www.eweek.com/c/a/Application-Development/Cobra-Language-Slithers-into-Open-Source/|Cobra Language Slithers into Open Source]]

Developed by the originator of the [[Webware]] [[WebFrameworks|Web framework]] for Python, focusing on the Common Language Runtime as implementation platform, provides type annotations, contracts, extended Python syntax.
