Put it in a file 'salaam.py':
# -*- coding: utf-8 -*-
#!/usr/bin/python3.1

def শালাম_বল(নাম):
   কথা = "শালাম {0}। কেমন অাছেন?".format(নাম)
   print(কথা)

অামি = 'রামি'
তমি = 'Carl'

if __name__ == '__main__':
   শালাম_বল(অামি)
   শালাম_বল(তমি)

***

Run the file:

$ python3.1 salaam.py
শালাম রামি। কেমন অাছেন?
শালাম Carl। কেমন অাছেন?
