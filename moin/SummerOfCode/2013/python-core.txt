= Project ideas for Python Core in GSoC 2013 =

1. Work on Email enhancements.  Possible projects:

    a. Add header-type-specific parsing for additional header types (Received,
       Message-Id, References) to the new 3.3 provisional policies.

    b. Work on support for [[http://tools.ietf.org/html/rfc6531|RFC 6532]] in the email package.

    c. Work on support for [[http://tools.ietf.org/html/rfc6531|RFC 6531]] in smtplib and/or smtpd.
