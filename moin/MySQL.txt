=== MySQL for Python ===

  URL:: http://sourceforge.net/projects/mysql-python
  licence:: GNU General Public License (GPL), Python License (CNRI Python License), Zope Public License
  platforms:: OS Independent 
  Python versions:: 2.4

== Supported Python Applications ==
 * ["Zope"]
 * A MySQL driver exists for ["PyDO"] (Python Data Objects)

== Pros ==
 * Many administration tools from third parties
 * Fast

== Cons ==
 * Only '''very''' basic SQL support (entry level SQL 92). Many interesting features (subqueries, views, triggers, ...) are missing.
 * Some [http://sql-info.de/mysql/gotchas.html gotchas] (documented but unexpected features)
