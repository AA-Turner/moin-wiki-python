#FORMAT rst

stdlib Clients
==============

Other parts of the stdlib rely on the email package for various services.
Right now this is just a list of import dependencies; this should be
expanded into a brief description of the stdlib use cases so we can
make sure we continue to support them properly.

=================  =====================================
Package            Email imports
=================  =====================================
http/client.py     email.parser, email.message
http/server.py     email.parser, email.message
urllib/request     email, email.utils (also mimetypes)
smtplib            email.utils
cgi                email.parser
pydoc              email.message
mailbox            email, email.message, email.generator
distutils/dist.py  email.message
=================  =====================================
