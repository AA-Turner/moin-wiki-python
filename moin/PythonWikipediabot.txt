The Python Wikipediabot Framework, also called pywikipedia is a set of bot scripts to do things on Wikipedia or other wikis using [[http:\\www.mediawiki.org|MediaWiki]]. It has been written by several people using the language Python. It is available form [[http://sourceforge.net/projects/pywikipediabot/|SourceForge]] or via SVN at http://svn.wikimedia.org/svnroot/pywikipedia/trunk/pywikipedia/ For help go to [[irc://irc.freenode.net/pywikipediabot|#pywikipediabot]] @ freenode server or use the [[http://lists.wikimedia.org/mailman/listinfo/Pywikipedia-l|pywikipediabot mailing list]]. A guide to pywikipedia can be found [[http://meta.wikimedia.org/wiki/Using_the_python_wikipediabot|here]]. There is also a wiki where is possible to test your scripts, [[http://botwiki.sno.cc/wiki/Main_Page|Botwiki]].
