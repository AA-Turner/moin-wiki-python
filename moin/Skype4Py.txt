[https://developer.skype.com/wiki/Skype4Py Skype4Py] is a [http://python.org Python] wrapper for the [https://developer.skype.com/Docs/ApiDoc Skype API]. It is platform independant, written completly in Python and reimplements the [https://developer.skype.com/Docs/Skype4COMLib Skype4COM]'s API in a pythonic way.
