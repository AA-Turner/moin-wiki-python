= Python Conferences =
The Python community tends to organize regional events around the world. Established events such as PyCon US and EuroPython have since been joined by other regional conferences, and Python often features prominently in various open source conferences.

This page aims to be the most complete list of Python conferences around the world and can be [[FrontPage#use|edited by the community]].

Want to organize an event in your community? Read [[AdvocacyWritingTasks/RunningAConference|our guide to running a conference]].

If you are planing a Python conference event, please check and register your event in the  [[https://pycon.org#calendar|Python Event Calendar]] to see whether there are any overlaps. See the PythonEvents page for schedule information.

=== Python at Open Source Conferences ===

Python has often featured in the following conference series and events:

 * [[https://confoo.ca/en|ConFoo Web Techno Conference]] (Canada) - features a Python track

 * [[https://fosdem.org/|FOSDEM]] (Belgium) - features a [[https://www.python-fosdem.org/|Python dev room]
 * [[https://conferences.oreillynet.com/oscon/|OSCON]] (USA) - features a Python track
 * [[https://www.osdc.com.au/|OSDC AU]] (Australia) - a very strong Python track
