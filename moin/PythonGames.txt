= Python Games =

  * [http://pygame.org Pygame.org] attachment:PyGameLogoTiny.gif Home of the PyGame module and a portal for PythonGames.
  * [http://pygame.org/shredwheat/solarwolf SolarWolf] an excellent action arcade game written with pygame.
  * [http://icculus.org/pyddr/ pyDDR] a high quality dance game created with pygame.
  * [http://www.oberhumer.com/opensource/pysol PySol] is an exciting collection of more than 200 solitaire card games.
  * [http://www.ikagames.com/ ika] is a game engine built on top of SDL with Verge as an ancestor.
  * [http://www.syays.com/ syays.com] using python to program games++ (a reference to the future).

See also: GameProgramming
