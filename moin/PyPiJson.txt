You can access JSON information about packages by using the URL format

  http://pypi.python.org/pypi/package_name/json

[[http://sphinx.pocoo.org/|Sphinx]] has support for putting an up-to-date Download link in your website using that JSON with one simple directive, for example:
{{{
.. pypi-release:: Roundup
   :prefix: Download
   :class: note
}}}
