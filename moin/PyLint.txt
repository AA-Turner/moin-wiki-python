[[http://www.logilab.org/projects/pylint|PyLint]]
is a [[static source analysis]] tool, available within such development tools [[http://pydev.sourceforge.net/pylint.html|as PyDev]], for instance.

= Availability =
* Currently runs on Python 2.4 and up
* An [[PyLint-3k|effort]] is getting underway to port to Python 3.x, while preserving Python 2.5 and up compatibility
