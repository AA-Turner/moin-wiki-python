= KOBRA - .NET for Python =

A .NET extension module for Python on windows. It works as a standard Python extension without requiring a modified Python interpreter.

The presentation at PyCON is attached [http://www.python.org/wiki/pub/PyConChetanGadgil/attachments/PyCON%20Presentation%20Slides.ppt here]

The full paper is here: [http://www.python.org/wiki/pub/PyConChetanGadgil/attachments/PyCON%20Paper%20on%20%22.NET%20for%20Python%22.htm .NET for Python Paper]



Details of the Project at [http://www.gadgil.net/DotNetWrapperForPython.htm KOBRA Project Page].
