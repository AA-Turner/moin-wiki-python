== Information about Python in languages other than English ==

Additionaly over just stating the link to your national python site you can also:
 * create a new page for your language as a WikiName (native words preferred) 
 * add the links at the end of the page to that page
 * translate the comments to your language
 * remove those links from this page
 * these pages don't have to be simple link collections. Use them for announcements, celebrate your local talent etc. If you find the time, create a full blown entry page into the world of Python

= Links from python.org =

These Links were/are on http://www.python.org/doc/NonEnglish.html. Hopefully there weren't any errors in the transcription.

== Bulgarian ==

 * The ''[http://python-bg.sourceforge.net/ Python Tutorial]'' is available in Bulgarian, thanks to [[MailTo(kaloian NO SPAM AT europe DOT com, Kaloian Doganov)]]. Please contact him if you'd like to help translate to Bulgarian!

== Chinese ==

 * [http://python.cn/ Python Chinese Forum and Mailing List]
 * [http://www.ibm.com/developerWorks/cn/linux/sdk/python/charm-24/index.shtml Python book reviews], translated into Chinese. The [http://www.ibm.com/developerworks/linux/library/l-pbook3.html English originals] were written by David Mertz.
 * [http://chinesepython.cosoft.org.cn/ ChinesePython] is a port of the Python language into Chinese (using Unicode). This is not just a translation of documentation, all keywords have been translated into Chinese, too. The project also has a Chinese [http://chinesepython.cosoft.org.cn/cgi-bin/moingb.cgi Wiki].
 * Wesley Chun's ''Core Python Programming'' is now [http://www.china-pub.com/computers/common/info.asp?id=3097 available in Chinese!] (China Machine Press, August 2001, 518 pp.)
 * The [http://coder.9ug.com/language/script/python/tut_tw/tut.html Python Tutorial] was translated by [[MailTo(hchou NO SPAM AT advertising DOT com, Harry Chou)]].
 * [http://diveintopython.org/cn/ Dive Into Python], a free Python tutorial for experienced programmers, has been translated into Chinese by limodou. Read it online or [http://diveintopython.org/download/cn/diveintopython.zip download it].

== French ==

Contributors are of course allways welcome.  The result of their efforts and current status can be found [http://frpython.sourceforge.net/docs/page.php3?id=1 here].

 * ''Learning Python'', by Mark Lutz and David Ascher, has now been translated into French by Olivier Berger, Sébastien Tanguy and Jérôme Kalifa.  It's in French bookstores as ''[http://www.editions-oreilly.fr/catalogue/lpython.html Introduction à Python]''.
 * ''[http://www.ceramiko.ch/python/main.html Tutoriel Python]'', a tutorial written by [[MailTo(j.tschanz NO SPAM AT romandie DOT com, Jérôme Tschanz)]].
 * [http://www.onera.fr/congres/jpf001/ First French Python Day] - May 28 1999, near Paris. Organized by PSA member ONERA.  Some resources:
    * [http://www.python.org/essays/ppt/jpf001.ppt Slides] of Guido van Rossum's talk (PowerPoint, in English).
    * [http://www.cs.rutgers.edu/~aaron/jpf/index.htm Slides] of Aaron Watters' talk (HTML, in English).  ([http://www.cs.rutgers.edu/~aaron/jpf/jpfarw.ppt PowerPoint], [http://www.cs.rutgers.edu/~aaron/jpf/network.py code])
    * [http://www.linux-center.org/articles/9906/python/ Slides] of Stefane Fermigier's talk (HTML, in  French).
    * [http://www.pns.cc/JPF001/abstract.htm Abstract of JPF001 slides] with links, by Kilian Golm and Christian Tismer.  They also have [http://www.pns.cc/JPF001/paris.ppt PowerPoint slides] available.
 * At [http://perso.club-internet.fr/olberger/python/python.html Olivier Berger's Python site] you'll find various packagings of a French translation of the Python tutorial, including:
     * [http://perso.club-internet.fr/olberger/python/doc/tut an online HTML version],
     * tar.gz and zip archives of it, and
     * dvi and postscript renderings.
 *  [[MailTo(fermigie NO SPAM AT math DOT jussieu DOT fr, Stefane Fermigier)]] had an article, [http://www.linux-center.org/articles/9812/python.html Présentation du langage Python], in the French computer magazine, ''Pogrammez!''. (10-Dec-1998)
