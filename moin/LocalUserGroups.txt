Python User Groups are sometimes called PIGgies, for Python Interest
Group meetings.  The term PIGgies was coined by the Bay Area PIGgies
group.

If there isn't an organized user group for your area listed on this page, check the list of [http://python.meetup.com/ Python Meetups] and see if there's a Meetup in your area.

= Starting and Running a User Group =

There are two mailing lists for organizers of user groups, if
you are interested in starting and running a user group:

(1) The [http://mail.python.org/mailman/listinfo/group-organizers Group-organizers] mailing list supports people starting and
running user groups.

(2) The [http://mail.python.org/mailman/listinfo/usergroup-advocacy Usergroup-advocacy] mailing list is looking at building tools and
resources useful for user groups.

Also, please consider posting news from your user group on the 
[http://python-groups.blogspot.com/ Python User Group Blog]!
