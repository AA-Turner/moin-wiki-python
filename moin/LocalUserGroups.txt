There about 400 Python user groups in the world.  You can find the link to them in two places:
 * The list below which is sorted by geographical region, country, and state in some cases.
 * A world map of [[http://python.meetup.com/|Python Meetups]] in meetup.com.  Enter your country and city to list the local group if any.

If there is not a Python user group in your area, please check [[StartingYourUsersGroup]] if you are interested in organizing one.

Python User Groups are sometimes called PIGgies, for Python Interest Group meetings. The term PIGgies was coined by the Bay Area PIGgies group.
