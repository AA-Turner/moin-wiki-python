[http://www.activestate.com/Products/Komodo/ Komodo] from ActiveState is a rich ["IDE"] for Perl, Python, PHP, Ruby and Tcl. It offers many integrated tools (i.e. code completion, code debugger, regular expressions toolkit, etc). ActiveState also introduced a free Komodo-based editor (EXPLAIN).
