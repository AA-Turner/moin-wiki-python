[http://www.activestate.com/Products/komodo_ide/ Komodo IDE] from ActiveState is a rich ["IDE"] for Perl, Python, PHP, Ruby, JavaScript, XML, HTML and Tcl. It offers many integrated tools (i.e. code completion, code debugger, regular expressions toolkit, etc). 

ActiveState also introduced an open-source, free Komodo-based editor, called [http://www.activestate.com/Products/komodo_edit/ Komodo Edit], which is less functional but available at no cost.
