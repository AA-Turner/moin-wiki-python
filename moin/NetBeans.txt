NetBeans IDE supports working with Python/Jython support since the NetBeans IDE 6.5 EA release for Python. More information can be obtained at http://wiki.netbeans.org/Python

The IDE currently features:
* Code Completion, Syntax highlighting etc.
* Debugging
* Test Runner
* Code Coverage

NetBeans is the best option for developing Python applications with its latest version 6.8
